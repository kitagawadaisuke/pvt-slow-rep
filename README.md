# Workout Tracker App

## 🚀 iPhoneでアプリを起動する方法

### 1回目だけ：Expo Goをインストール
- iPhoneのApp Storeで「Expo Go」を検索してインストール

### 毎回の起動手順

1. **VSCodeでターミナルを開く**
   - VSCodeの下部にあるターミナル画面をクリック

2. **以下のコマンドを入力**
   ```bash
   npm run tunnel
   ```

3. **ターミナルに表示されるQRコードをスキャン**
   - iPhoneで「Expo Go」アプリを開く
   - カメラでQRコードをスキャン
   - アプリが起動します

**これだけです！**

---

## 📝 その他のコマンド

```bash
npm start        # 通常モード（同じWi-Fi必要）
npm run tunnel   # トンネルモード（Wi-Fi不要） ← おすすめ
```
