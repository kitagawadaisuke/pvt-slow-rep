# Product Hunt

## App Name
SlowRep

## Tagline
No ads. No subscriptions. Your workout data is yours.

## Description

SlowRep is a workout tracker built for slow repetition training.

I was frustrated with existing fitness apps — ads during rest periods, paywalls to view old history, no way to track tempo or variations. So I built my own.

Key Features:
- Track variations (narrow, wide, diamond) and tempo (4-2-2) per set
- Built-in metronome for maintaining consistent rep speed
- Interval timer with voice countdown
- Save workout menus and load them with one tap
- Calendar view for training history
- Monthly/yearly stats dashboard
- JSON export for AI analysis (feed it to ChatGPT/Claude)
- 100% local storage — no account, no cloud, no tracking

Built with React Native (Expo) + Claude Code in about a month.

Free. No ads. No subscriptions. Ever.
